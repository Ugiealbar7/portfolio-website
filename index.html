// 01 — tentang saya

Siapa Saya?

Saya Muhamad Ugi Albar, mahasiswa Sistem Informasi di Universitas Pembangunan Jaya (UPJ) yang memiliki passion kuat di bidang rekayasa perangkat lunak, desain antarmuka, dan tata kelola teknologi informasi.

Selama masa studi, saya aktif membangun aplikasi web menggunakan Laravel, Node.js, dan MySQL, serta terlibat dalam berbagai project akademik mulai dari manajemen pengetahuan konstruksi hingga sistem layanan smart city.

Saya juga tertarik pada bidang IT Audit menggunakan kerangka kerja COBIT 4.1, dengan pengalaman langsung melakukan audit unit TIK dan menganalisis tingkat kematangan proses teknologi informasi organisasi.

Saat tidak coding, saya aktif melakukan riset dan penulisan jurnal ilmiah, khususnya terkait Software Quality Assurance dan metodologi pengujian perangkat lunak.
